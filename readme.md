# Registro de Ponto (MVP)

Aplicação Web empresarial de Registro de Ponto com dois perfis:
- FUNCIONARIO: registra ponto e consulta seu relatório
- RH: consulta relatórios de qualquer funcionário

## Stack
- Java + Spring Boot
- PostgreSQL
- JWT (AuthN/AuthZ)
- JPA/Hibernate
