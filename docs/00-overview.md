# Visão Geral

## Objetivo
Sistema de Registro de Ponto para uso empresarial.
O sistema permite que funcionários batam ponto e que relatórios sejam consultados pelo próprio funcionário e pelo RH.

## Perfis de usuário
- FUNCIONARIO
    - Registrar ponto (entrada/saída)
    - Consultar relatório próprio por período
- RH
    - Consultar relatório de qualquer funcionário por período

## Escopo do MVP
Inclui:
- Autenticação via JWT
- Registro de ponto (ENTRADA/SAIDA) com horário do servidor
- Relatórios por período (listagem de registros)

Fora do escopo (por enquanto):
- Ajustes manuais de ponto (abono)
- Aprovação de ponto
- Geolocalização / biometria / foto
- Controle avançado de escalas/turnos
- Exportação PDF
